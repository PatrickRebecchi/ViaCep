# Busca CEP com Java

Projeto em Java para consumo da API ViaCEP utilizando HttpClient e Gson
para converter respostas JSON em objetos Java.

## Tecnologias
- Java 24+
- HttpClient
- Gson
- API ViaCEP

## Estrutura
- modelos → classes de dados
- service → regras de negócio e integração com API
- principal → ponto de entrada da aplicação

## Como executar
1. Execute a classe PrincipalCep
2. Digite um CEP válido (somente números)

## Exemplo
Digite o CEP:
02079010

CEP: 2079-010
Logradouro: Rua Itala
Cidade: São Paulo - SP
